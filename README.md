## Repository Moved / Unified Codebase

We have unified the codebases for all four of our works on robust conformal prediction into a single repository:

[RobustConformalPrediction](https://github.com/UoC-tail/RobustConformalPrediction)

This unified repository now contains the implementations and reproduction code for our robust conformal prediction methods, including CAS, BinCP, RCP1, and FLoR.

Please refer to the unified repository for reproducing our results and for any future use of the code.

Thank you.
